# 🌐 Online Portfolio

> [!NOTE]
> This repository contains the source code, documentation, and development history of my online portfolio.

---

## 🎯 Purpose

This repository is used to track the development and improvement of my online portfolio.

It helps me keep a cleaner workflow, better documentation, version control, and a more organized development process.

It also works as a public reference for recruiters or developers who want to review the code behind my portfolio.

> [!IMPORTANT]
> Modern technologies are part of an efficient development workflow, especially for managing time, reducing repetitive tasks, and improving productivity.
> However, they do not replace human reasoning, professional judgment, technical criteria, or the logic behind the work performed.

---

## 📌 Project Overview

This project is my personal online portfolio.

It presents:

| Section | Purpose |
|---|---|
| 👤 About | Brief explanation of who I am |
| 🧠 Skills | Technical skills and tools |
| 🎓 Background | Studies, experience, and learning path |
| 💻 Projects | Academic, personal, and technical projects |
| 🌎 Languages | English and Spanish portfolio content |

---

## 🛠️ Technologies & Tools

| Category | Tools |
|---|---|
| Frontend | React, TypeScript, Vite |
| Styling | Tailwind CSS |
| Version Control | Git, GitHub |
| IDE | IntelliJ IDEA |
| Workflow | Git, GitHub, documentation, version control |
| Deployment | Static web hosting |
> [!NOTE]
> The initial UI draft was created with Figma Make as a starting point.
> The final structure, content, styling, fixes, responsiveness, and implementation decisions are manually reviewed, customized, and maintained.
